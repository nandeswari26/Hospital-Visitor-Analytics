# 🏥 Hospital Visitor Analytics & ML Project

## 📌 Overview
This project analyzes hospital visitor data to monitor patient trends,
wait times, and satisfaction scores using Power BI and Python.

## 📊 Dashboard Features
- Total Visitors KPI
- Average Wait Time
- Satisfaction Score
- High Wait Risk %
- Department-wise Analysis
- Time Intelligence (YoY Growth)

## 🤖 Machine Learning
- Built classification model to predict high wait time cases
- Used Logistic Regression and Random Forest
- Evaluated using accuracy, precision, recall

## 🛠 Tools Used
- Power BI
- DAX
- Python (Pandas, NumPy, Scikit-learn)
- Matplotlib / Seaborn
