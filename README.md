# 🏥 Hospital Visitor Analytics & Predictive Modeling

## 📌 Project Overview

This project focuses on analyzing hospital visitor data to monitor patient flow, operational efficiency, and satisfaction levels. 

The solution combines:

- 📊 Power BI Dashboard (Interactive Analytics)
- 📈 DAX-based KPI Tracking
- 🤖 Machine Learning for Risk Prediction

The goal is to transform raw hospital data into actionable insights and predictive intelligence.

---

## 🎯 Business Objectives

- Monitor total visitor trends over time
- Analyze department-wise patient distribution
- Identify high wait time risk cases
- Measure patient satisfaction performance
- Predict operational risk using machine learning

---

## 📊 Dashboard Features (Power BI)

### 🔹 Core KPIs
- Total Visitors
- Average Wait Time
- Average Satisfaction Score
- Year-over-Year Growth %

### 🔹 Risk Indicators
- High Wait Time %
- Low Satisfaction %
- High Wait & Low Satisfaction %

### 🔹 Segmentation Analysis
- Visitors by Department
- Visitors by Age Group
- Weekday vs Weekend Analysis
- Monthly & Yearly Trends

---

## 🤖 Machine Learning Component

A classification model was developed to predict:

- High Wait Time Cases (>60 minutes)
OR
- Low Satisfaction Risk

### 🔹 Steps Performed:
- Data Cleaning & Preprocessing
- Feature Engineering
- Train-Test Split
- Model Training (Logistic Regression, Random Forest)
- Model Evaluation (Accuracy, Precision, Recall, F1-Score)

## 📈 Key Insights

- Higher wait times significantly reduce patient satisfaction.
- Emergency department shows elevated wait-time risk.
- Visitor trends indicate peak weekday traffic.
- Operational bottlenecks identified through department-level analysis.
